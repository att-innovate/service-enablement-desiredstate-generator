Serverspec-generator
============

[![npm version](https://badge.fury.io/js/serverspec-generator.svg)](https://badge.fury.io/js/serverspec-generator)


Development
----------

### How to build

#### requirements

- nodejs && npm
- make
- ruby 2.2 or newer
- Go 1.5 or newer

```sh
$ npm i
$ make depends
$ make
```

### Lint check

```sh
$ make lint
```


### Run example

```sh
$ make browserify
```

and open example/index.html by browser
